# Shared UI Components

This app does not currently define a shared `src/components` or `src/components/ui` layer.

All visible UI is implemented directly inside page files:

- `src/pages/Home.tsx`
- `src/pages/Guidian.tsx`
- `src/pages/Chunkworld.tsx`

There are no reusable primitive components such as `Button`, `Card`, `Input`, `Badge`, or `Dialog` in the current codebase.
